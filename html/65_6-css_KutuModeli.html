KUTULARIN YASLANMASI(FLOAT)

bazı durumlarda herhangikutunun sola veya sağa yaslanması gerekir bu kutular veya nesneler bi resim fiv tagı gibi bi bölüm veya tablo olabilir kutuların yaslandırıması için float özelliği kullanılır.float özelli ile yaslandırılan her kutu mümkün olan en sol ya da en sağ tarafa yaslanır diğer nesneler float özelliği ile yaslanan kutunun etrafını sararlar eğer nesnelerin kutu etrafını sarmasını istemiyorsanız clear özelliğini kullanırsınız. normalde tüm html elemanlar bulundukları sıraya göre alt alta gösterilir.

artık float etkin olmamasını istiyorsak ilk bitişinden itibaren ilk elementin css özellikleirine aşağıdaki gibi bir css özelliği verilebilir bu şekilde kalan tüm taglar olduğu alt alta gösterilir.burada örnek olarak boş bir div tagı oluşturuldu ve bu tag sadece clear işlevi için kullanıldı bu şekilde diğer tüm css biçimlendirmeleride kendi işlevlerinde devam edecek sadece istediğimiz zaman div tagını class ile birlikte kullanarak clear işlemi gerçekleştirebiliriz.

clear için örnek div tagı 

< div class="clear" >< /div >
clear için örnek css kodu 
clear için örnek css kodu
.clear{
cleat:both;
