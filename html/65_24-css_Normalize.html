normalize css dosyasına yazılmış olan tanımlamalar görüntülenen sayfanın tüm tarayıcılarda aynı biçimde görünlenmesi için yazılmış kodlardır.Bu sayfanın görünümündeki margin ve paddin özelliklerinden bu anlaşılabilir. bizim yazacağımız css dosyasının normalize.css dosyasının altında link edilmesinin sebebi ise tarayıcı önce hangi css dosyası link edildiyse onu yorumlar ondan sonra link edilen bir css dosyası varsa sırayla onu yorumlar yani en son link edilen css dosyası her zaman geçerli kalır.biz yapacağımız değişiklikleri normalize.css dosyasının özelliklerinin üzerine yapmış olacağız ve iki dosyada birbirinden etkilenmeyecek ve yapıları bozulmayacak.
