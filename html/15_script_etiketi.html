SCRIPT ETİKETİ

Vbscript JScript veya java Script kodlarını yazılacağı alandır. gerektiğinde body etiket sınırları içerisinde de kullanılabilir.
       

etikete ait bazı özellikler
      1. Async =boolean tipindedir. sıra beklemeksizin eş zamanlı olarak script kodlarının çalışacağını belirler dahili script kodlarında kullanılamaz harici başka çağrılan scriplerde kullanılır. sayfa yüklenirken kodlar çalışmaya başlar.
      2. Defer =booleann tipindedir.sayfa yüklendikten sonra script kodları icra edilmeye başlanır.harici script dosyalarında kullanılır.
      3. Src =harici bir script dosyasının yolunnu belirtmek için kullanılır.
      4. Type =bir komut dosyasının internet medya türünü belirtir.
