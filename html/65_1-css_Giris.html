CSS GİRİŞ DERS 1

css her html komutunu bir nesne ya da kutu olarak görmektedir. her nesnenin de değişik özellikleri ve her özelliğin de alabileceği farklı değerlerin vardır css komutları 3 kısımdan oluşur.

  • html komutu (selector)
  • özellik (property)
  • değer (value)

tipik bir css komutu aşağıdaki gibidir.

söz dizimi seçici {komut1;}{komut2;}
örnek p {color:red;font-size:20px;}

p tagına css uygulamak için 

p{
color:blue;
font-size:20px;
}
şeklinde bir kodlama biçimi yapılmalıdır ve kod okunabilirliğide artırılmış olur.

hx taglarına css uygulamak için şöyle bir kullanım yapılabilir. 

h1,h2,h3,h4,h5,h6{

font-family:arial;
}

eğer tüm p taglarının ve resim açıklamalarının rengini değiştirmek istiyorsak aşağıdaki gibi bir kullanım yapılabilir.

p, figcaption{
color:red;
}

eğer yazmak istediğimiz css kodları bir id ile tanımlanmış bir nesneye aitse aşağıdaki # işareti ile id değeri yazılarak süslü parantezler içerisine css kodları yazılır.

#id-değeri{
css kodları;
}

eğer bir css komutunu bir class nesnesine uygulamak istiyorsanız class isminin başına nokta koyarak süslü parantezler içerinde css kodlarımızı yazabiliriz.

 
.class-ismi{
css komutları;
}

eğer css komutlarının içerisine açıklama satırları eklemek istiyorsanız.

CSS KODLARI NEREYE YAZILIR?

CSS kodları üç farklı bölgeye yazılabilir.

  1. Inline : satır içi (html tagının içi) 
    < p style="color:red;"> BU ÖRNEK BİR PARAGRAF
  2. Internal:dahili sayfanın head tagının içine
    3. < html >
< head >	
< style >

p{
color:red;
font-size:15px;
}

< /style >
< /head >


< body >
< /body >

< footer >
< /footer >
< /html >
  3. External:harici (proje klasörü içinde html sayfası dışında bir dosya)

    4. html dosyasının yanında style.css adında bir boş bir dosya oluşturulur ve hangi html dosyasında kullanılmak isteniyor css komutları o dosyalara link verilir link verme işlemi html dosyasının head etiketlerinin içerisine yapılır. 

    < html >
< head >	

< link rel="stylesheet" type="text/css" href="/css-dosyaları/style.css" >
< /head >

< body >
< /body >
< footer >
< /footer >
< /html >

css kodlarını external biçim de yazmak daha iyidir html kodlarından ayrı olması html kodlarını yazarken daha iyi odaklanmamızı ve daha verimli çalışmamızı sağlar.style.css dosyasının içerisinde css kodları yazarken herhangi bir selector değişikliği yoktur.

HANGİ CSS KODU ÖNCELİKLİ?

öncelikli olan css kodları inline kodlardır.ikinci olarak öncelikli olan internal css kodlarıdır.üçüncü olarak öncelikli olan css kodları external css kodlarıdır.
