}

.son{
color:red;
}

color:red;
}
h1 taglarının içindeki adı egik olan class ı kapsar 
nav#buton{
color:red;
}

h1,h2,h3,h4,h5,h6{
color:red;
}
h1...h6 taglarını kapsar 
body,strong,em{
color:red;
}
body , strong ve em taglarını kapsar 
nav p{
color:red;
}

.banner{
color:red;
}

color:red;
}
class ismi banner olan p taglarını kapsar

color:red;
}
class ismi banner olan tagların içindeki class ismi menu olan tagları kapsar. 
.banner>.menu ul{
color:red;
}
class ismi banner olan tagların içindeki class ismi menu olan tagların içindeki ul taglarını kapsar. 
.banner>.menu ul li{
color:red;
}
class ismi banner olan tagların içindeki class ismi menu olan tagların içindeki ul taglarının içindeki tüm li taglarını kapsar. 
nav.menu{
color:red;
}
nav etiketlerinin içindeki class ismi menu olan tagları kapsar.

color:red;
}

body>p:first-child{
color:red;
}

body>.p:last-child{
color:red;
}
body tagları içindeki son p tagını kapsar.

color:red;
}

color:red;
}

div + p{
color:red;
}

KOMUT ADIYLA SEÇİCİ KULLANIMI

eğer bir tag içerisindeki tüm taglara bir özellik uygulanmak isteniyorsa komut adıyla seçiçi kullanımı uygulanır.

 
p{
color:green;
}

TEKİL SEÇİCİ KULLANIMI

eğer bir tag içerisindeki bir çok aynı tagdan sadece birine bir özellik uygulanmak isteniyorsa tekil seçici kullanılır.Bunun için istenen taga id değeri verilir ve o id değeri üzerinden istenen özellik css kodları ile uygulanır. 


#id{
color:green;
}

SINIF CLASS SEÇİCİLER

sınıd seçicileri istediğimiz kadar html tagına uygulayabiliriz.tekil seçiciler yerine class seçicilerin kullanılması daha uygun olabilir. 

.arialyaz{
font-family:arial;
buradaki kodlar tüm arialyaz sınıflarının içerisinde ve alt taglarında geçerli olacaktır.
}

ÖZELLİK SEÇİCİLER

sadece belli bir özelliğe sahip html elemanlarının seçilmesini sağlar.Örneğin bir form içinde bulunan ve type özelliği text olan bütün elemanları yani metin kutularını seçebiliriz.

seçici özellik açıklama
[özellik] 

[text]{
color:red;
}
type özelliği olan tüm html elemanlarını seçilir.
[özellik=değer] 

[type=text]{
color:red;
}
type özelliği text olan tüm html elemanları seçilir.
[özellik~=değer] 

[title~="web"]{
color:red;
}
title özelliği içinde web kelimesi geçen elemanları seçer
[özellik|=değer] 

[title|="web"]{
color:red;
}
title özelliği web ile başlayan elemanları seçer
[özellik^=değer] 

a[href^="https"]{
color:red;
}
href özelliği https ile başlayan tüm a taglarını seçer
[özellik$=değer] 

a[href$=".pdf"]{
color:red;
}
href özelliği .pdf ile biten tüm a taglarını seçer.
[özellik*=değer] 

a[href=*="pdf"]{
color:red;
}
href özelliği içinde pdfmetni olan tüm a taglarını seçer

tabloda gösterilen özellikler:

  • belli bir özelliğe sahip olan
  • özelliği belli bir değere sahip olan
  • özelliği belli bir değerle başlayan
  • özelliği belli bir değerle biten
  • özelliği belli bir metin olan html komutlarnı seçilip css komutu uygulabilir.

aşağıdaki tabloda özellik seçiciler ile ilgili örnekler ve açıklamaları bulabilirsiniz .

css seçiciaçıklama 
input[type]
< input > elemanı içinde type özelliğine sahip olanları seç 
input[type="text"]
< input > elemanı içinde type özelliği text olanları seç 
a[taget=_blank]
< a > tagı içinde hedefi target değeri _blank olanları seç 
img[title~="kedi"]
< img > tagındaki title özelliği içinde kedi kelimesi olanları seç 
div[class*="menu"]
class özelliği içinde menu metni olan div elemanını seç
