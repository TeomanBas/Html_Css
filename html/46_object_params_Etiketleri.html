< object > etiketi

İnternet tarayıcıları tarafından desteklenmeyen özellikle java appletleri,Activex pdf ve flash gbi nesneleri sayfaya yerleştirmek için kullanılır.
eklentilere parametre geçirmek için param etiketi kullanılır.özellikleri aşağıdadır.

  1. DATA:Nesnenin kullanılacağı kaynağın url sini belirtir.
  2. FORM:nesnenin ait olduğu bir veya daha fazla formu belirtir.
  3. HEİGT/WİDTH:nesnenin sırayla yüksekliğini ve genişliğini belirler
  4. NAME:nesne için isim belirler
  5. TYPE:data özelliğinde belirtilen veri ortam türünü belirtir.
  6. USEMAP:nesne ile kullanılacak bir istemci taraflı görüntü yerleşiminin adını belirtir.


< param > etiketi

< object > etiketiyle sayfaya eklenmiş nesnelerin özelliklerini belirlemek için kullanılır.sonlandırmalı değildir.object etiketlerinin içerisinde kullanılır.
özellikleri aşağıdadır.

  1. NAME:Parametrenin adını belirler
  2. TYPE:Medya tipini belirler
  3. value:parametrenin alacağı değeri belirler.
  4. valuetype:parametrenin alacağı değerin tipini belirler.
